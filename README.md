Material Design 3 (Expressive) の哲学を WordPress テーマに昇華させた、次世代のクリエイティブ・プラットフォーム。

## v0.3 "Evolution" - 2026.4.23

### 🌌 ユニバーサル・レスポンシブ・エンジン
- **Fluid Scaling:** `clamp()` を用いた宇宙規模のスケーリング。iPhone 5 (320px) から 8K プロディスプレイまで破綻なくカバー。
- **24px Sanctuary:** 右端の余白を「聖域」として全解像度で死守。

### 🏎️ 240Hz / 8K "Overdrive" 最適化
- **GPU Acceleration:** GSAP に `force3D: true` を適用し、GPU レイヤーへ強制転送。
- **Adaptive Visuals:** 高DPI環境を自動検知し、`backdrop-filter` の強度を動的に調整。
- **Low Latency:** `requestAnimationFrame` によるイベントデバウンスで、入力ラグを最小化。

### 🧩 Hegemony (覇権) カスタムブロック
- **スマート・ソートテーブル:** 文字列・数値を自動判別する軽量 Vanilla JS ソート機能。
- **演出付き投票ユニット:** 投票時に Canvas パーティクルが弾ける視覚体験。
- **チェック連動型メディアラベル:** GSAP による Chips 形式の滑らかな出現演出。
- **万能メディア・ブリッジ:** Apple Music, Spotify, Steam, ニコニコ動画を M3 秩序（角丸 16px, Elevated）へ統合。

## インストール
1. `wp-content/themes/node` に配置。
2. 管理画面より有効化。
3. 必要に応じて `functions.php` に Gemini API キーを設定。

# リリースノート - Node テーマ

## [0.2.0] - 2026.4.22
### 変更点
- **テーマカラーのアップデート:** `color_script.js` で抽出したオレンジ系（Luminouscore）のテーマカラーをライト・ダーク両モードに適用し、統一感のあるデザインに調整。
- **ファビコンの追加:** `Luminouscore.svg` をテーマのファビコンとして設定。
- **バージョン更新:** テーマおよび関連ファイル（package.json 等）のバージョンを 0.2.0 へ更新。

## [0.1.8] - 2026.4.19
### 変更点
- **バグ修正と構文エラーの解消:** `functions.php` の構文エラー（Parse error）を修正。
- **PHP 8 互換性向上:** 配列アクセス時の厳格な型チェックに対応し、PHP 8 以降で発生していた警告（Warning）を解消。
- **テーマ切り替えボタンの改善:** ダーク・ライトモード切り替えロジックを修正。ライトモード時は太陽、ダークモード時は月のアイコンを表示するように変更。
- **RSS ボタンの改善:** ホバー時に RSS カラー（オレンジ）になるようにスタイルを調整。
- **日付表示の高度化:** 1日以内の投稿を「◯時間◯分前」、7日以内を「◯日前」と表示し、それ以降は日付を表示するようにロジックを更新。
- **カテゴリ表示の最適化:** 記事個別ページでは全てのカテゴリを表示し、一覧カードでは4つ以上のカテゴリを省略（+N表示）するように調整。
- **AI生成ラベルの刷新:** ラベルの文言を「生成されたメディアを含みます」に変更し、`auto_awesome` アイコンを追加。
- **バージョン更新:** テーマ全体のバージョンを 0.1.8 へ更新。

## [0.1.6] - 2026.4.18
### 変更点
- **アイコン読み込みの修正:** Material Symbols, Google Fonts, Font Awesome の読み込みを WordPress の標準エンキューシステムに集約し、テーマ全体でアイコンが表示されない問題を解消。
- **フォント設定の集約:** `functions_fonts.php` を作成・有効化し、テーマで使用する全ての外部フォントとアイコンを一元管理。
- **コードのクリーンアップ:** `header.php` の手動フォントリンクを削除し、パフォーマンスと保守性を向上。
- **CSS の最適化:** `.material-symbols-outlined` クラスに `font-family` を明示的に指定。
- **バージョン更新:** テーマおよび関連ファイルのバージョンを 0.1.6 へ更新。

## [0.1.4] - 2026.4.12
### 変更点
- **ヘッダーレイアウト:** 
  - PCビューでハンバーガーメニューを非表示化。
  - タイトルを中央配置し、固定ページメニューをその直下に移動。
  - 検索窓をRSSの左へ配置し、クリック時に展開されるインタラクティブな仕様に変更。
  - 検索窓の展開アニメーションを滑らかに調整。
- **スポンサーラベル:**
  - Material 3 Expansive スタイルを採用（丸みのある形状、高コントラスト）。
  - ライト・ダークモード両対応のカラーデザイン。
  - 記事個別ページの上部ヒーロー画像右上に絶対配置。
  - 投稿編集画面で、ラベルの表示文言とマウスホバー時の説明文をカスタム入力可能に変更。
- **記事カード:**
  - 日付表示を右上に最適化。
  - Material Symbols のカレンダーアイコンを追加。
  - ラベル要素のホバー時の挙動を改善し、不要なカーソル表示を削除。
- **セマンティック HTML:**
  - 記事ループ内で `article` タグを使用するように構造を統一。
  - 日付表示に `time` タグを採用。
- **バグ修正:**
  - `single.php` における制御構造の構文エラーを解消。
  - 記事個別ページ内の重複していたカテゴリ表示を削除。
  - ヘッダーボタンの不要な黒い枠線を削除。

## [0.1.3] - 2026-04-11

- **高コントラストダークモード:** ダークモード時の背景色をより深くし、テキストの視認性とカードの境界線のコントラストを大幅に改良。

### 変更点

- **ミニマル Spotlight:** SPOTLIGHT セクションをアイコンなしのカラーパネルデザインへ最終調整。

### バグ修正
- **カテゴリ除外:** Spotlight カテゴリとそのすべての子カテゴリが、一覧や個別記事から確実に非表示になるようロジックを強化。
- **バージョン更新:** 全アセットのバージョンを 0.1.3 へ更新し、キャッシュ問題を解消。
