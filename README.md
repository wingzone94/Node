Material Design 3 (Expressive) の哲学を WordPress テーマに昇華させた、次世代のクリエイティブ・プラットフォーム。

## インストール
1. `wp-content/themes/node` に配置。
2. 管理画面より有効化。
3. 必要に応じて `functions.php` に Gemini API キーを設定。

# リリースノート - Node テーマ

## [0.5.0] - 2026.4.26
### 変更点
- **記事詳細ヘッダーの刷新:** `single.php` のヘッダーを「オーバーラップ・カード」形式に再構築。カテゴリーラベルをタイトル下の中央に配置し、情報の対称性と安定感を向上。
- **インタラクティブ・スポンサー吹き出し:** `single.php` において、タイトルをホバーした際のみ現れる「フローティング・バブル（吹き出し）」形式のスポンサー表示を実装。
- **ミニマルなスポンサー表示:** `index.php`（記事カード）ではインフォメーションアイコンのみの表示とし、ホバー時に左側へツールチップを展開するスマートな仕様に変更。
- **FAB カラーの固定化:** 右下の追従ボタン（最上部に戻る・コメント・目次等）のカラーをブランドカラー（#FF9900）に固定し、カテゴリー色に左右されない一貫した操作感を提供。
- **ツールチップ・インテリジェンス:** 画面端や他のFABとの衝突を検知し、自動的に左側へシフトする回避ロジックを実装。
- **ユニバーサル・デイトバッジ:** 記事カードの日付をカード右下に固定。アイキャッチ画像の有無に関わらず、すべての記事で日付が確実に確認できるよう改善。
- **安定性の向上:** 記事カードの PHP 構文エラーを解消し、構造をクリーンアップ。日付ラベルの「見切れ」問題も配置調整により完全解決。

## [0.4.0] - 2026.4.26
### 変更点
- **PWA (Progressive Web App) 完全対応:** `manifest.json` および `sw.js` (Service Worker) を新規実装。Chrome, Edge, Android, iOS において「アプリとしてインストール」が可能になり、オフライン性能が向上。
- **Safari 15+ ブラウザバー色の最適化:** `theme-color` メタタグの出力優先度を最大化し、HTML 背景色のハックを適用。Safariのアドレスバーをテーマカラーのオレンジ (#FF9900) に確実に同期。
- **トップページ・ループの構造刷新:** 最新4記事を「⚡ LATEST」、それ以降を「🕒 ARTICLES」としてセクション分け。重要度に基づいたメリハリのあるレイアウトを実現。
- **404ページの新規実装:** ページが見つからない場合のカスタムエラーページを Material 3 準拠で作成。
- **タグのハッシュタグ化:** タグ表示を「#タグ名」の形式に変更し、SNSのようなモダンな外観へ。
- **タイポグラフィとビジュアルの磨き込み:** `optimizeLegibility` の有効化、ロゴのグラデーション化、記事本文の行間 (`1.8`) とカーニングの微調整による可読性向上。
- **コメント欄のカラー完全統一:** 残存していた青・紫系のリンクやボタン色をすべてオレンジ (#FF9900) に統一。
- **SEO/AMP 最適化:** JSON-LD 構造化データの自動生成機能を `functions.php` に実装し、リッチリザルト対応を強化。
- **コピーボタンのUX改善:** コピー成功時のテキストが2行に折り返されないようCSSを修正し、文言から絵文字を削除。

## [0.3.2] - 2026.4.25
### 変更点
- **AI 要約アコーディオンの実装:** 記事カード一覧に `<details>` ベースの折りたたみ式 AI 要約を追加。Gemini API を消費せず、DB に保存されたメタデータのみで高速動作。
- **M3 カラーシステムの動的化:** 「投稿個別カラー ＞ カテゴリカラー ＞ アイキャッチ抽出色 ＞ デフォルト」の優先順位で Material 3 カラーを動的に生成するロジックを実装。
- **利用規約準拠の外部埋め込み:** Apple Music, Google Maps, Spotify の埋め込みコードを公式仕様（iframe src 直接入力）に準拠させ、規約を遵守した設計にリライト。
- **商品カードの一本化:** Amazon・楽天市場の同時表示に対応した `[product_card]` ショートコードを実装し、旧機能を統合。
- **フッター領域の装飾:** ナビゲーションエリア（このブログについて等）を Secondary Container 色で色分けし、視認性と意図的なゾーニングを強化。
- **表示件数の最適化:** スクロール負荷低減のため、PC: 16件（4×4）、Mobile: 8件 に調整。
- **AI 開発モデルの拡張:** 本バージョンより、Claude Sonnet 4.6 および Claude Opus 4.6 を併用して開発。
- **バージョン更新:** テーマ全体のバージョンを 0.3.2 へ更新。

## [0.1.8] - 2026.4.19
### 変更点
- **バグ修正と構文エラーの解消:** `functions.php` の構文エラー（Parse error）を修正。
- **PHP 8 互換性向上:** 配列アクセス時の厳格な型チェックに対応し、PHP 8 以降で発生していた警告（Warning）を解消。
- **テーマ切り替えボタンの改善:** ダーク・ライトモード切り替えロジックを修正。ライトモード時は太陽、ダークモード時は月のアイコンを表示するように変更。
- **RSS ボタンの改善:** ホバー時に RSS カラー（オレンジ）になるようにスタイルを調整。
- **日付表示の高度化:** 1日以内の投稿を「◯時間◯分前」、7日以内を「◯日前」と表示し、それ以降は日付を表示するようにロジックを更新。
- **カテゴリ表示の最適化:** 記事個別ページでは全てのカテゴリを表示し、一覧カードでは4つ以上のカテゴリを省略（+N表示）するように調整。
- **AI生成ラベルの刷新:** ラベルの文言を「生成されたメディアを含みます」に変更し、`auto_awesome` アイコンを追加。
- **バージョン更新:** テーマ全体のバージョンを 0.1.8 へ更新。

## [0.1.6] - 2026.4.18
