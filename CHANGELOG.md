# CHANGELOG

Node テーマの更新履歴です。

## [1.2.0] - 2026.07.14

### シリーズ（連載）機能
- **記事をシリーズにまとめる機能を新設**: `node_series` タクソノミーで記事を連載としてまとめ、記事内に目次・前後ナビ・カード上のバナー（現在回/全話数）を表示。シリーズ共通のプライマリカラー（term meta）と記事ごとの色上書き（post meta）の2段階設定に対応（優先順位: 記事上書き > シリーズ共通色 > ブランドオレンジ `#FF9900`）。登録上限は10件/シリーズ。シリーズ内の記事が1件以下の場合は記事上に一切表示しない。
- 管理画面「シリーズ内の表示順序」メタボックスで登録先シリーズ・表示順をプルダウン選択。無効な選択（上限到達済み・使用済みの表示順）はAjaxのリアルタイム再計算でUI上選択不可に。ブロックエディタ／クラシックエディタの双方で動作。
- シリーズ term の削除時に投稿側の表示順メタを自動で後片付け（記事別カラー上書きは意図的に保持）。

### ブログカード / 埋め込み
- **ブログカードを oEmbed パイプライン方式へ刷新**: 自サイト記事URLがカード化されない不具合を修正。`the_content` 優先度11の正規表現スキャン（`node_auto_blogcard()`）を廃止し、`oembed_dataparse`（解決成功時）と `embed_maybe_make_link`（解決失敗時）にフックする方式へ変更。自サイト記事・他のWordPressサイト・一般ブログURLを統一して `m3-blogcard` デザインで表示。
- **X(Twitter) と YouTube は対象外**とし、WP標準のネイティブ埋め込みをそのまま維持。
- **埋め込みブロック `node/embed` を新設**: エディタで X / YouTube / Google マップのURLを貼ると `node/embed` に自動変換し、エディタ内にプレビューを表示。フロント出力は従来と同一（`node_special_embed()` へ委譲）。
- 既存記事の埋め込みは投稿メタ `_oembed_*` のキャッシュが切れるまで旧デザインのまま表示されます（再保存で即時反映）。

### 記事ヒーロー
- **PCヒーローを縦2カラム構成へ再設計**: 左に核心（カテゴリ→タイトル→メタ）、右に補足（読了→著者→開示バッジ→目次）。1000px以下では従来同様の1カラムへ自動で折りたたみ。
- **目次をヒーローへ統合**（ネイティブ `<select>` ベースのプルダウン、マルチページ記事はページ単位でグループ化）。モバイルはヒーロー通過後に目次FAB＋パネルを解放（`body.is-hero-passed`）。
- **読了バッジを簡素化**: 「約◯分で読めます」＋「約N,NNN文字」チップに変更。読了時間は**550字/分の固定換算**、長さ判定は**絶対文字数**（〜1,500短い / 〜3,000やや短い / 〜6,000標準 / 〜10,000やや長い / 以上長い）に改定（旧「サイト平均比」の相対換算による異常値を解消）。ランク連動の信号色をやめブランドオレンジ固定に。
- **AI開示チップを静音化**: オレンジ塗りのCTA風からアウトラインの情報チップへ。アイコン色を可読色に固定（動的カラーで灰青に化ける不具合の修正）。
- アイキャッチを幅基準の縮尺で全体表示に統一（16:9固定枠・レターボックスを廃止）。画像なしヒーローは上詰めに。
- 投稿日を日付アーカイブへのリンクに（追記日は表示のみ）。コメントは1件以上のときだけカウンターを表示。

### 記事カード
- **カード全面クリック**（stretched link 方式）。カテゴリピル等は個別操作を維持。
- グリッドの行内高さを統一（`align-items: stretch`）。
- **追記日を表示**（同年なら「追記 7/11」の短縮表記で投稿日と1行表示）。判定は `node_get_post_modified_display()` に共通化。
- AI・スポンサーの開示バッジをアイキャッチ上のオーバーレイからカテゴリ行右端へ移動（衝突解消）。

### Node Library
- **ライブラリを新設計**（ストアフロント一覧＋個別ページ）。タイプフィルターはプリティURL（`/node-library/game/` 等）でのサーバーサイド絞り込み。「この作品に触れた記事」の逆引きにも対応。
- 記事ヒーローからライブラリへのクイックリンクを追加。

### アーカイブ / ナビゲーション
- **日付アーカイブを新設計**（`date.php`）: 日別グループのタイムライン型リスト＋期間セレクタ＋前後ナビ。
- **404ページを新設**（`404.php`）: 検索フォームとホームへの導線を備えた軽量デザイン。noindex / nofollow を付与。
- **ライター別アーカイブ導線とSNSピルを追加**: WRITER INFO に記事一覧チップ（ライター名の横）と、ブランドカラー付きSNSピル（X / GitHub / YouTube / Facebook / Instagram / note の6種プリセット）。モバイル（≤600px）では44px円形アイコンに短縮。
- **フッターに公式SNSサークルを常設**（X・Discord）。モバイルでヘッダーから導線が消える問題を解消。
- モバイルのボトムバーを廃止し、トップFABのみに整理。

### 修正 / 内部
- 記事見出しの下線色を記事アクセント連動の `color-mix` へ変更（動的カラーで背景に同化して消える不具合の修正）。連続する見出しが同一行に並ぶ不具合も修正。
- `node_resolve_redirect`（blogcard.php）の `sslverify=false` を撤回（既定 true）。
- `node-series` にバージョン比較による rewrite flush を追加（`node-library` と同型）。パーマリンク再保存なしで `/series/{slug}/` が解決。
- Node Flow の `Scroller.php` に `whitelist_query_vars()` を追加。
- 検索モーダルの未実装項目（人気/コメントソート・読了目安・AI生成チップ）をUIから退避。ソート値を実装済みの newest/oldest に修正。
- **プラグインブランドを統一**: Plugin Name を全て「Node ○○」に統一、Version を全プラグイン 1.2.0 に統一（Author は「Luminous Core Teams」で固定）。

## [1.1.7] - 2026.07.11

- **公開後数時間での反映に対応**: 追記日の記録・設定ゲートを公開後24時間から**3時間**に短縮（保存時の自動記録・エディタ「更新日表示」パネルとも）。あわせて、手動メタ（自動記録含む）がある場合は**公開日と同じ日付でも「追記」を表示**するよう変更（当日中の訂正開示のため）。エディタパネルの日付下限も公開日当日から選択可能に。

## [1.1.6] - 2026.07.11
### 記事ヒーロー（モバイル）
- **お知らせブロック（node/notice）を先行実装（v1.2から移植）**: お知らせ / 注意 / 重要 / 補足の4種のコールアウトブロック。記事への訂正掲載などに使用可能。ブロックカテゴリ「Node」・エディタスクリプト（notice部分のみ抽出、node/embed等のv1.2専用ブロックは含まない）・フロント配色CSS（_notice.css、テーマstyle.cssにバンドル）を含む。保存時にHTML確定（render_callbackなし）のためv1.2移行時も既存投稿はそのまま表示される。
- **追記日の表示を追加（v1.2と共通の仕組み）**: 記事ヒーローのメタ情報に、更新日が投稿日と異なる場合のみ「追記 YYYY/MM/DD」を表示（updateアイコン付き・リンクなし）。手動上書きメタ `_node_manual_modified_date`（ブロックエディタの「更新日表示」パネル・公開24時間後から設定可・保存時の自動記録付き）を優先し、未設定時はWP標準の更新日を使用。v1.2の実装をそのまま移植。
- **読了バッジ下の空洞を修正**: モバイル幅のサムネイル付き記事で、ヒーローカードのボトム余白（ヘッダーインナー32px＋オーバーラップ32px）が二重に積まれ、読了バッジの下に約80pxの空白ができていた崩れを修正（ライター報告・Chrome/モバイルで顕著）。インナー側の余白をモバイルで畳み、オーバーラップ側を20pxに調整（空白 約80px → 約36px）。PC表示は変更なし。

## [1.1.0] - 2026.06.13
### パフォーマンス・外観・検索 UX
- **Font Awesome 廃止**:
    - ヘッダー・シェアボタンをインライン SVG / Material Symbols へ移行。`scripts/verify-icon-classes.mjs` で残存 FA クラスを監査可能に。
- **フォント読み込み最適化**:
    - Material Symbols を `display=block` でレンダーブロッキング読み込みし、アイコンの FOUT（文字が一瞬見える現象）を抑制。
    - 本文フォント（Inter / Noto Sans JP）は非同期 + `display=swap` のまま維持。
- **外観モード（カラーモード）**:
    - 日の出・日没による自動切替を主軸に、シングルクリックで手動反転、ダブルクリック（PC）/ 長押し（モバイル）で OS 同期へ復帰。
    - 同期時に「外観設定をシステムと同期しました」通知（PC: 右上ピル / モバイル: 下部ポップアップ）。
- **検索 UI**:
    - キーワードクリア（`header.php` インライン）と霧散エフェクト。ブラウザ標準の検索 × と重複しないよう非表示。
    - 検索起動時は RSS / SNS / テーマ等を隠し、PC ではロゴを残したまま検索バーをボタン分だけ拡張。起動中は虫眼鏡を非表示にし ← で閉じる。
    - `Escape` で検索を閉じる（PC）。
- **PWA / ホーム画面追加**:
    - iOS 向け `apple-touch-startup-image`（10 解像度）と `apple-touch-icon`（PNG）を追加。オレンジ背景 + ブランドロゴ。
    - `scripts/generate-pwa-splash.mjs` でスプラッシュ画像を再生成可能。
- **タブレット UA 判定**:
    - `node_is_tablet_ua()` を追加。iPad / Android タブレット等で表示モード切替ボタンをヘッダーに出力。
- **ブランド表記**:
    - テーマ Author とフッターコピーライトを **Luminous Core Team** に統一。
- **ドキュメント**:
    - `AI-AGENTS.md` を追加（開発で利用する AI エージェント / モデル一覧）。
