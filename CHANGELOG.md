# CHANGELOG

Node テーマの更新履歴です。

## [1.1.2] - 2026.06.21
### Node Library
- **タブ分類バグ修正**: ストア名が「PS Store」のリンクが「PC」タブに表示されていた問題を修正。PlayStation 系（PS Store / PSN 含む）を「コンソール」タブへ正しく分類。
- **手動タブ分類**: 各ストアリンクに「表示タブ」(自動判定 / PC / スマホ・タブレット / コンソール) を追加し、自動分類を手動で上書き可能に。
- **Gemini 連携対応**: 情報取得（Gemini）の応答にも `category` を含めるようプロンプト・パースを拡張し、取得結果から表示タブを自動設定できるように。
- **ストア追加**: Amazon Appstore（スマホ・タブレットタブ / ブランドカラー #FF9900）と GeForce NOW（PC タブ / NVIDIA グリーン #76B900）に対応。許可ドメインと Gemini プロンプトにも追加。

## [1.1.0] - 2026.06.13
### パフォーマンス・外観・検索 UX
- **Font Awesome 廃止**:
    - ヘッダー・シェアボタンをインライン SVG / Material Symbols へ移行。`scripts/verify-icon-classes.mjs` で残存 FA クラスを監査可能に。
- **フォント読み込み最適化**:
    - Material Symbols を `display=block` でレンダーブロッキング読み込みし、アイコンの FOUT（文字が一瞬見える現象）を抑制。
    - 本文フォント（Inter / Noto Sans JP）は非同期 + `display=swap` のまま維持。
- **外観モード（カラーモード）**:
    - 日の出・日没による自動切替を主軸に、シングルクリックで手動反転、ダブルクリック（PC）/ 長押し（モバイル）で OS 同期へ復帰。
    - 同期時に「外観設定をシステムと同期しました」通知（PC: 右上ピル / モバイル: 下部ポップアップ）。
- **検索 UI**:
    - キーワードクリア（`header.php` インライン）と霧散エフェクト。ブラウザ標準の検索 × と重複しないよう非表示。
    - 検索起動時は RSS / SNS / テーマ等を隠し、PC ではロゴを残したまま検索バーをボタン分だけ拡張。起動中は虫眼鏡を非表示にし ← で閉じる。
    - `Escape` で検索を閉じる（PC）。
- **PWA / ホーム画面追加**:
    - iOS 向け `apple-touch-startup-image`（10 解像度）と `apple-touch-icon`（PNG）を追加。オレンジ背景 + ブランドロゴ。
    - `scripts/generate-pwa-splash.mjs` でスプラッシュ画像を再生成可能。
- **タブレット UA 判定**:
    - `node_is_tablet_ua()` を追加。iPad / Android タブレット等で表示モード切替ボタンをヘッダーに出力。
- **ブランド表記**:
    - テーマ Author とフッターコピーライトを **Luminous Core Team** に統一。
- **ドキュメント**:
    - `AI-AGENTS.md` を追加（開発で利用する AI エージェント / モデル一覧）。

## [1.0.3] - 2026.06.09
### SPOTLIGHT 過去特集カタログ・単一記事レイアウト調整
- **SPOTLIGHT 専用ページ（`/spotlight/`）**:
    - 記事一覧・ページネーションを廃止し、過去の特集テーマ（子カテゴリ）カードのみを表示するカタログページに変更。
    - ホームの「すべて見る」から `/spotlight/` へ誘導。
    - `/category/spotlight/` は `/spotlight/` へ 301 リダイレクト。
- **単一記事の整列**:
    - タイトルカードと本文画像の左右揃えをモバイル・折りたたみ・デスクトップ各幅で統一。
    - ブラウザ既定の `figure` 余白とヒーローカードの bleed を解消。
- **モバイル余白の最適化**:
    - Single / HEADLINES / SPOTLIGHT 向け fluid トークンを追加し、セクション間の過大な余白を抑制。
- **テーマアップデート・バージョン表示**:
    - Luminous Settings / フッター / アップデート確認で `node_get_theme_version()` により同一の `style.css` Version を参照。
    - GitHub からの自動更新で Linux 本番でも `Node/` フォルダを正しく解決するよう ZIP 展開パスを修正。
- **SPOTLIGHT URL（`/spotlight/`）**:
    - 本番でリライト未フラッシュ時に 404 になる問題を修正（`request` / `template_redirect` フォールバック、rewrite v5 再フラッシュ）。

## [1.0.2] - 2026.06.08

- **アーカイブヘッダーの整理**:

- **単一記事タイトルカード**:
    - タイトルカードのグラデーションと枠線を削除し、フラットな配色に統一。

## [0.7.0] - 2026.05.04
### 追加・変更点 (Material 3 UI Optimization Update)
- **ヘッダー・レイアウトの刷新**:
    - 構造を全面的に整理し、ロゴとアクションボタン（検索、RSS、テーマ、表示切替）の配置を完璧に最適化。
    - 不要な重複リンク（ナビゲーション）を排除し、コンテンツへの集中を高めるクリーンなヘッダーへ。
- **フローティング・アクション・スタックの再設計**:
    - 「最上部に戻る」ボタンをスタックの最上位に配置。出現しきい値を 300px に短縮し、より早い段階でのナビゲーションを可能に。
    - ボタンの ID 存在チェックを強化し、一部のボタンがないページでも正常に動作する堅牢な JS ロジックへ更新。
- **「明るさ」制御アイコンの導入**:
    - テーマ切り替えアイコンを `brightness_6` に固定。モードに関わらず「明るさ・外観の調整」を想起させる直感的なデザインへ。
- **UI のクリーンアップ**:
    - 記事詳細ページのタイトルカード上にあった不要なアクセントラインを削除。
    - 静的なトップページでも時計/挨拶機能が正しく動作するように修正。
    - タブレット端末（iPad/Android Tablet）のみ「PC/タブレット表示切り替え」ボタンを表示する UA 判定を導入。

## [0.6.3] - 2026.05.02
### 追加・変更点 (Mobile & 404 Not Found Update)
- **モバイル検索モーダルの UX 刷新**:
    - モバイルでの操作性を考慮し、フィルターのタブ切り替えを廃止。直感的に全項目へアクセスできる「シングルスクロール・ストリーム構造」へ統合。
    - 不要な余白を徹底的に排除し、Material 3 Expressive のデザインをモバイル向けに最適化。
- **タグサジェスト機能の導入**:
    - 検索バーに入力と連動して既存のタグ候補を表示するサジェスト機能を実装。モバイルでの検索利便性を大幅に向上。
- **404 ページのゲーミフィケーションとアートギャラリー**:
    - マインクラフト、フォートナイト、Windows BSOD、Material 3 等をテーマにした 6 種類のランダム・エフェクトを実装。
    - 隠し機能：タイトルエリアを 3 回連続クリックすることで、全テーマを鑑賞できる「404 Art Gallery」が発動。
- **Material 3 "Paper" UI**:
    - Material 3 Expressive 404 テーマを、背景画像から「デジタル・ペーパー」をコンセプトにしたレイヤー構造の UI デザインへと刷新。
- **モバイル UX の最適化**:
    - モバイル環境でのフォートナイト版出現頻度を制御（1/10）し、デバイスに馴染むテーマを優先。
    - 全 404 テーマの完全レスポンシブ化（clamp() による可変フォントサイズ）。
    - 記事メタ情報（並び順と読了目安）の不自然な空白など、モバイルでの細かなレイアウト崩れを徹底修正。
- **セキュリティと著作権**:
    - アートワークの右クリック・ドラッグ禁止によるコピーガードを実装。
    - 実在の会社名（Mojang AB, Epic Games 等）に基づいた静的な権利表記を画面右下に配置。
